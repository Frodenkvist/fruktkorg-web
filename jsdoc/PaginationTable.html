Name Type Description

Properties
Name Type Description
columns Array.<Object> A list of column objects.
Properties
Name Type Description
name String Displayed in table header.
key String Links row content to the right column. should be the same on both column and row.
comparator function Optional, a comparator for sorting the column. If not provided, the column is not sortable
rows Array.<Object> The items to render. The properties on the fields is expected to be the same keys defined in columns.
itemsPerPage Integer Initials how many items that are displayed on the page.
noItemsText String Optional, Adds a message if there are no objects to render.
sortByKey String Optional, The key for the column that should be used as the default sorting. Requires a comparator to have been set.

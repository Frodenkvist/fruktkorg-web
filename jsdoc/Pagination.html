Class: Pagination

Pagination(props)

A generic component that can be used where pagination is needed. Renders a pagination bar.

Constructor

new Pagination(props)

Parameters:
Name Type Description
props Object Holds the information to render the component
Properties
Name Type Description
items Integer The amount of items
itemsPerPage Integer The amount of items to display per page
currentPage Integer The number for the current page to be displayed
onPageChange function Optional, An callback function called when changing the current page. Called with two parameters, pageNumber and itemsPerPage. Use this to update other components that displays items.
onItemsPerPageChange function Called when the amount of items per page changes. Called with two parameters, pageNumber and itemsPerPage. Use this to update other components that displays items.
